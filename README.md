# Regalitos

**Regalitos** es una app para crear y compartir listas de regalos de bebé. Los invitados pueden ver la lista, reservar regalos y marcar cuáles han comprado, todo sin necesidad de registrarse.

## Arrancar

```bash
# Instalar dependencias
npm install

# Iniciar en desarrollo
npm run dev
```

Necesitas un archivo `.env` con las credenciales de Firebase. Pídelas al administrador del proyecto.
